# Cafeteria Management Dashboard

A clean, modern, and responsive web application designed for cafeteria management. This project demonstrates professional front-end practices, including modular JavaScript, CSS Grid systems, and smooth UI transitions.

## 🚀 Features

- **Single Page Feel:** Uses URL hash-based navigation to switch dashboard sections without page reloads.
- **Responsive Grid System:** The menu is displayed using a CSS Grid layout that automatically adapts to any screen size.
- **Sticky Navigation:** Features a fixed top navbar and a sticky sidebar for consistent access to navigation links.
- **Modern UI/UX:** Built with CSS variables for easy theme management and `@keyframes` for smooth fade-in animations.
- **State Management:** A centralized JavaScript `App` object manages the application state and active navigation highlights.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Modern layout techniques (Flexbox & Grid) and Custom Properties (Variables).
- **Vanilla JavaScript:** Clean, modular code without the overhead of heavy frameworks.

## 📂 Project Structure

- `index.html`: The main landing page.
- `dashboard.html`: The interactive dashboard interface.
- `style.css`: Global styles and theme variables.
- `dashboard.css`: Dashboard-specific grid layouts and sidebar styling.
- `script.js`: Core application logic and navigation handling.
